JSP开发基础

JSP页面元素构成

JSP页面元素由静态内容[HTML标签]、脚本元素、指令、动作、注释等组成

JSP基本语法

  1. 脚本元素

  2. 内置对象

    JSP内置对象是由JSP容器定义好的一些常用对象，脚本元素可以把他们作为预定义变量直接使用，从而访问请求和web应用的数据。

    JSP提供了9个内置对象分别是
    • request
    • response
    • out
    • session
    • application
    • exception
    • page
    • pageContext
    • config

  3. 指令元素

    JSP指令包括page、include、taglib指令
    • <%@ page=””>
    • <%@ include=””>
    • <%@ taglib=””>
      需要注意的是他们只在当前的JSP页面有效

  4. 动作元素

  • jsp基本动作元素
    • <jsp:incude>
    • <jsp:forward>
    • <jsp:param>
    • <jsp:plugin>
    • <jsp:params>
    • <jsp:fallback>
  • 与javaBean组件有关的元素
    • <jsp:useBean>
    • <jsp:setProperty>
    • <jsp:getProperty>
  • jsp2.0新增动作元素
    • <jsp:invoke>
    • <jsp:doBody>
